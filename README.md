 Project Overview

A **clean, responsive static webpage** designed for a nature travel brand called "Nature Horizon". This project demonstrates modern HTML5 and CSS3 techniques including semantic structure, CSS Grid for macro layouts, Flexbox for micro components, and fully responsive design.

 Features

- ✅ Semantic HTML5 - Uses `<header>`, `<main>`, `<section>`, `<article>`, `<footer>`, `<nav>`
- ✅ CSS Grid - For macro page layout (cards grid, stats section, footer)
- ✅ CSS Flexbox - For micro components (navigation, buttons, newsletter form)
- ✅ Fully Responsive - Works on desktop, tablet, and mobile devices
- ✅ No JavaScript - Pure HTML & CSS only
- ✅ External CSS - DRY principle, no inline styles
- ✅ Gradient Backgrounds - Modern visual appeal
- ✅ Hover Effects - Smooth transitions on cards and buttons
